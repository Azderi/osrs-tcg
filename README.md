# OSRS TCG

A **RuneLite** plugin that adds a light collectible card experience on the side of the grind. Credits are obtained by playing the game, spend them on **booster packs**, and grow your card collection.

Cards come in **different rarities**. Pulls can be **foil**. There is a **sidebar panel** for the shop, packs, tuning, and stats, a **pack opening** flow with reveal UI and optional sounds, and a **collection album** to browse what you own or are missing.

The plugin features party plugin integration that allows you to trade cards with other players.
